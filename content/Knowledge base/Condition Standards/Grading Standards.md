# Grade A
- Brand New condition
- Open Box condition
- Like New condition
- No visible wear
- Minor battery wear (90%+)

# Grade B
- Good Condition
- Light visible wear
- Minor battery wear (85%+)

# Grade C
- Fair Condition
- Moderate wear
- Moderate battery wear (70%+)

# Grade D
- Damaged Condition
- Heavy wear
- Heavy battery wear (under 70%)

# Grade R
- Recycle only
- Device does not power on
- Device is obsolete
