## Cracked Glass
```
the device’s glass panel is cracked, though the LCD remains fully functional.
```
## Cracked LCD
```
The LCD panel is cracked and no longer functions, preventing the display from producing a usable image.
```
## Backlight Bleed
```
the display contains noticeable backlight bleed, resulting in uneven brightness and light leakage along portions of the screen. This defect affects overall display quality and visual consistency.
```
## Broken Hinge
```
the unit’s hinge has detached from the chassis, creating structural instability and posing a risk of further damage with continued use.
```
## Delamination
```
the LCD shows signs of delamination, with visible bubbles along the lower edge of the display. This defect affects the screen’s cosmetic and functional integrity.
```
## Discoloration
```
The device’s display exhibits uneven color uniformity across the screen, with noticeable variations in brightness and color tones between different areas of the panel.
```
## Ghosting
```
The device’s LCD panel exhibits ghosting, resulting in visible image retention and display artifacts that impact normal viewing performance.
```
## Hairline Fracture
```
the display contains a hairline fracture, which poses a risk of further damage and affects overall condition.
```
## Line defect
```
The device’s display contains visible line defects that interfere with normal viewing and indicate a fault within the panel or display cable.
```
## Nonfunctional
```
The display has been damaged and is no longer capable of producing an image. As a result, the screen is considered not to fully light up.
```
## Nonfunctional Backlight
```
The device’s display backlight fails to power on, rendering the screen unusable. While the system may still boot and function internally, the absence of backlight illumination prevents normal visibility and operation. This issue significantly impacts usability and requires repair or replacement of the display assembly to restore proper functionality
```
## Nonfunctional Section
```
the display is defective and contains a section that does not display, meaning the screen does not fully light up.
```
## Uneven Light Diffusion
```
The device’s display exhibits uneven light diffusion, resulting in inconsistent brightness across the screen.
```
## [[White Spot]] 
```
The display contains a visible white spot.
```
