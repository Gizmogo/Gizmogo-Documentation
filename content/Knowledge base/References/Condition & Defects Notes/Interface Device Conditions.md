## Broken Switches
```
The unit has one or more broken key switches, affecting keyboard functionality and requiring repair or replacement to restore proper operation.
```
## Broken Keycap
```
one or more of the key caps on the keyboard is chipped, cracked, or broken.
```
## Input Issue
```
The keyboard is defective with multiple keys that either inconsistently or fail to register inputs.
```
## Missing Parts
```
The device’s keyboard is incomplete and is missing essential components such as key caps, switches, rubber gaskets, or a combination of these parts.
```
# Touchscreen
## Ghost Input
```
The unit’s touchscreen exhibits ghost inputs, registering unintended touches that interfere with normal operation. This issue negatively impacts usability when using touch or other interface inputs.
```

## Nonfunctional Section
```
The touchscreen contains a non-functional section that does not respond to touch input, limiting normal operation and usability.
```

# Trackpad
## Input issue
```
The Trackpad is defective and inconsistently or fails to register inputs.
```
